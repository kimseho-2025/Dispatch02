## 🚀 빠른 테스트 방법

### ✅ 방법 1: 로컬 파일 직접 열기 (추천!)

**Windows:**
```
1. 파일 탐색기 열기 (Win + E)
2. 프로젝트 폴더로 이동
3. index.html 파일을 Chrome으로 드래그 앤 드롭
```

**또는:**
```
1. Chrome 열기
2. Ctrl + O
3. index.html 선택
```

**주소창 확인:**
```
file:///C:/Users/.../Dispatch02/index.html
```

## ❌ 에러 발생 시

### "showPage is not defined" 에러

**원인:** 브라우저가 이전 버전 캐시 사용

**해결:**
1. **로컬 파일 직접 열기** (위 방법 1)
2. **완전 캐시 삭제:**
   - Ctrl + Shift + Delete
   - "전체 기간" 선택
   - 쿠키 + 캐시 삭제
3. **시크릿 모드로 테스트:** Ctrl + Shift + N
4. **GitHub Pages 배포 대기:** 5분 후 재시도

### "IndexedDB 오류"

**원인:** 브라우저가 IndexedDB 미지원 또는 차단

**해결:**
- Chrome/Edge 최신 버전 사용
- 시크릿 모드 아닌 일반 모드 사용
- 브라우저 설정 → 쿠키 허용 확인
